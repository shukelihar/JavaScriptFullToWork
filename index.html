16.Учимся работать с системой контроля версий Git и с сервисом GitHub

1) git init - сразу инициализируем

2) git config --local user.name "Ihar"
Локально или глобально
Второй email
git config --local user.email "shukelihar@gmail.com"
Пример: первый раз через --global
git config --global user.name "Ihar"
git config --global user.email "shukelihar@gmail.com"

3) git status проверить какие файлы

4) git add -A - добавляет все файлы в отслеживание (git add namefile.css добавит только файл 1 *.css все с данным расширением)

5) git commit -a -m"first commit" - забрасываем
